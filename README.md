# Bài tập: Trí tuệ nhân tạo trong Quản trị Công nghệ

## Giới thiệu

Đây là các bài tập thực hành liên quan đến bản đồ số, phân tích dữ liệu không gian và ứng dụng AI trong quản trị công nghệ.
Các bài tập tập trung vào việc xử lý dữ liệu địa lý, trực quan hóa bản đồ và xây dựng các mô hình phân tích – dự đoán trong thực tế.

---
## Công nghệ sử dụng

* Python
* Folium (hiển thị bản đồ)
* GeoPy (geocoding)
* GeoPandas (xử lý dữ liệu địa lý)
* OSMnx (mạng giao thông)
* NetworkX (đồ thị & tìm đường)
* Scikit-learn (machine learning – phân cụm, dự đoán)
* Numpy, Pandas...
---
